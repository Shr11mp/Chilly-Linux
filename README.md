# ❄️ Chilly-Linux

My personal, reproducible NixOS configuration repository managed via Nix Flakes and Home Manager. 

> 🏗️ **Status:** This configuration is currently being built and actively developed.

---

## 🖥️ Desktop Environments

This configuration features a dual-desktop setup tailored for different workflows:
*   **Hyprland (Main):** A dynamic tiling Wayland compositor utilized as the primary, highly customized daily driver.
*   **GNOME (Default):** A stable, full-featured desktop environment maintained as a reliable fallback and conventional workspace.
